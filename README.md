# HealthBridge

HealthBridge is an AI-powered healthcare accessibility platform designed to bridge the gap between individuals and essential healthcare resources.

Built during a hackathon, the platform combines symptom analysis, health risk assessment, mental wellness support, and emergency healthcare discovery into a single user-friendly experience.

## Problem Statement

Millions of people delay medical consultation due to:

* Lack of awareness
* Difficulty assessing symptoms
* Limited access to healthcare information
* Mental health stigma
* Inability to quickly locate nearby medical services

HealthBridge addresses these challenges through an accessible digital healthcare assistant.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/healthbridge.git
```

Navigate into the project:

```bash
cd healthbridge
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
